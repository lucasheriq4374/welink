## 🧭 What is welink?

welink is a local AI app for WeChat chat data analysis. It helps you view chat trends, find active friends, explore group chat patterns, and build word clouds from your chat history.

It runs on your own computer. You keep your data local while the app helps you read it in a clear way.

## ✨ What you can do

- See friend activity rankings
- View group chat profiles
- Generate word clouds from chat text
- Track emotion trends over time
- Ask AI to analyze chat content
- Work with local chat data on Windows, MacOS, or Docker

## 💻 Before you install

Use a Windows PC with enough free space for your chat data and app files.

- Windows 10 or later

## 🧩 Common issues

### The app does not open

Try these steps:

- Download the latest release again
- Make sure the file finished downloading
- Unpack the archive if you downloaded a `.zip`
- Run the file from a normal folder like Desktop or Downloads

### The app cannot read my chats

Check:

- The folder path is correct
- The data export is complete
- The chat files are not damaged
- You have selected the right source folder

### The app feels slow

Large chat sets can take time to load. Try:

- Closing extra apps
- Using a folder with fewer files first
- Waiting for the first scan to finish

## 🗂️ Folder tips

A clean folder setup helps the app work better.

Keep app files and chat data separate if you can.
